# Roadmap

This document outlines the planned direction of IssueScout.

---

# Version 0.2.0

## Features

* GitHub GraphQL API integration
* Improved repository scanning performance
* Advanced issue filtering
* Better pull request relationship detection
* Configurable scanner settings

## Developer Experience

* Additional integration tests
* Improved API documentation
* More detailed logging
* Performance benchmarking

---

# Version 0.3.0

## CLI

* `issuescout scan`
* `issuescout analyze`
* `issuescout repo`

## Backend

* Request caching
* Rate-limit handling
* Background jobs
* Persistent scan history

## API

* Pagination
* Sorting
* Filtering
* Versioned endpoints
